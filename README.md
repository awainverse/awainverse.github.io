# Nadine Bradbury's Website

This repository hosts the [website](https://nadinebradbury.github.io/) for Nadine Bradbury, a postdoc in the [Subotnik Group](https://subotnikgroup.chemistry.princeton.edu/) at [Princeton Chemistry](https://chemistry.princeton.edu/).

## Template

This website is based on the [Hugo](https://github.com/gohugoio/hugo) template [hugo-website](https://github.com/pmichaillat/hugo-website) by [Pascal Michaillat](https://pascalmichaillat.org/).
